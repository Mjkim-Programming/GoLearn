# GoLearn
This is a simple Project that I am doing to Learn go.

## Spec
### Package Hello
#### SayHello
```go
import(
    "github.com/Mjkim-Programming/GoLearn/hello"

    "fmt"
)

func main() {
    fmt.Println(hello.SayHello("GaegulDev"))
}
```
is..
```bash
$ Hello, GaegulDev!
```

#### SayBye
```go
import(
    "github.com/Mjkim-Programming/GoLearn/hello"

    "fmt"
)

func main() {
    fmt.Println(hello.SayBye("GaegulDev"))
}
```
is..
```bash
$ Bye, GaegulDev!
```
